📚 Wikipedia Searcher (Python)

A simple Python program that lets you search Wikipedia from the terminal and get quick summaries of any topic.

🚀 Features

Search any topic from Wikipedia

Get a short summary instantly

Simple command-line interface

Lightweight and beginner-friendly Python code

🛠 Requirements

Install the required library before running the program:

pip install wikipedia

🎯 Purpose

This project is made for beginners learning Python and web information retrieval.

It demonstrates:

Python loops

User input

Using external libraries

Simple information extraction
